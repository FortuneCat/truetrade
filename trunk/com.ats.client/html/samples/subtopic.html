Working with MySQL

By default, TrueTrade comes configured to use Apache Derby, a lightweight, Open Source database that Sun will soon be shipping with Java by default. It is a 2Mb JAR file so is easy to deploy, but some users will want the extra speed and power that a bigged database can provide. A popular choice is the Open Source database engine MySQL.

To configure TrueTrade to run with MySQL, open the database preferences window and set the following properties:

  • User name = guest
  • Password = guest
  • Provider = com.mysql.jdbc.Driver
  • URL = jdbc:mysql://localhost/ats?enable-named-pipe&socketFactory=com.mysql.jdbc.NamedPipeSocketFactory
